  <p align="center">Um simples microserviço construído em  <a href="https://github.com/nestjs/nest" target="_blank">NestJS</a> com CRUD de usuários, recuperação de senhas por email, autenticação com token JWT e uso do Swagger para melhor manipulação dos endpoints. A imagem Docker para este projeto pode ser encontrada no <a href="https://hub.docker.com/r/douggoncalves/nestjs-crud-api" target="_blank">link</a> </p>
  
## 💻 Instalação

```bash
$ git clone https://github.com/DougGoncalves/nestjs_api
$ docker-compose up -d
```
## 📍 Rodando a Aplicação Localmente

```bash
# watch mode
$ npm run start:dev

```
## ⚙️ Testes

```bash
# testes unitários
$ npm run test:watch

```
